# Fashion MNIST Image Classification using CNN

## Project Overview
A Convolutional Neural Network model built to classify 
fashion items from the Fashion MNIST dataset.

## Result
Test Accuracy: 90.55%

## Dataset
- Name: Fashion MNIST
- Training images: 60,000
- Test images: 10,000
- Image size: 28x28 grayscale
- Classes: 10 clothing categories

## Classes
0 - T-shirt
1 - Trouser
2 - Pullover
3 - Dress
4 - Coat
5 - Sandal
6 - Shirt
7 - Sneaker
8 - Bag
9 - Ankle boot

## Model Architecture
- Conv2D Layer (32 filters)
- MaxPooling Layer
- Conv2D Layer (64 filters)
- MaxPooling Layer
- Flatten Layer
- Dense Layer (128 units)
- Output Layer (10 units)

## Technologies Used
Python, TensorFlow, Keras, Matplotlib, NumPy

## How to Run
1. Open notebook in Google Colab
2. Run all cells
3. Dataset downloads automatically

## Conclusion
CNN model successfully classifies fashion items 
with 90.55% accuracy on test data.
